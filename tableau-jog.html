Custom Parameter Control

If you're tired with the same slider parameter control in your Tableau Dashboard here is a nice trick to use any kind of JS widget to change your parameter. The trick that your embedded HTML frame can access the tableau javascript object from inside the dashboard. This is simple done by accessing window.top.tableau from your HTML frame.

The method works from Tableau Public and Tableau Server as well.

Code example

// The trick:  parent.parent in vizportal and window.top for tableau public 
getTableau = function() {
  return window.top.tableau || parent.parent.tableau;
};

getCurrentViz = function() {
  return getTableau().VizManager.getVizs()[0];
};

getCurrentWorkbook = function() {
  return getCurrentViz().getWorkbook();
};

getCurrentWorksheet = function() {
  return getCurrentViz().getWorkbook().getActiveSheet().getWorksheets()[0];
};
